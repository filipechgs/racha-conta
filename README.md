# Racha Conta

Um pequeno app web para dividir valores e calcular total de forma simples.

## O que ele faz

- Registra itens com nome e valor
- Suporta operações: soma, subtração, multiplicação, divisão e porcentagem
- Exibe histórico de lançamentos
- Calcula total em tempo real
- Permite dividir o total entre pessoas
- Suporta copy/share e instalação como PWA

## Estrutura do projeto

- `index.html` — página principal
- `css/style.css` — estilos do app
- `js/app.js` — lógica do app
- `sw.js` — service worker para cache PWA
- `assets/icons/` — ícones usados pelo PWA
- `assets/manifest/manifest.json` — manifesto do PWA

## Como usar

1. Abra `index.html` em um servidor local ou em um ambiente web.
2. Adicione itens, escolha operação e registre.
3. Use o botão `Dividir` para calcular quanto cada pessoa deve pagar.
4. Instale como PWA em dispositivos compatíveis.

## PWA

O app já está preparado para ser instalado como PWA em Android, iOS e desktop.
